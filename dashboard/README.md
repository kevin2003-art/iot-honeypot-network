# IoT Honeypot Dashboard

A military-style threat intelligence dashboard for the Healthcare IoT Deception Honeypot Network.

## How to run

Start the honeypot and launch the dashboard:

    bash scripts/launch_dashboard.sh

Or update data only:

    bash scripts/update_dashboard.sh

## Tabs

- OVERVIEW: attack stats, top passwords, usernames and commands
- ATTACKERS: IP registry with threat levels
- CREDENTIALS: full credential dictionary used by attackers
- COMMANDS: every command with risk classification
- CHARTS: hourly attack timeline and login breakdown
- WORLD MAP: attacker geolocation by country
- LIVE FEED: real time attack event feed

## Data files

- dashboard_data.json: main stats generated by generate_dashboard_data.py
- chart_data.json: chart data generated by generate_chart_data.py
- geo_enriched.json: geolocation data generated by geolocate_ips.py
